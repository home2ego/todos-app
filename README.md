# Todos App

"Todos App" is a single-page application that allows users to add and manage todos on both the client and server.

### [Todos Preview](https://todos-web.pages.dev/)

## Technologies Used

- React.js
- TS
- JS
- Fetch, REST API
- Sass (SCSS)
- Bulma
- Font Awesome
- React Transition Group

## Features

>  **Dual Storage**:
路 Todos are stored on both client and server.

>  **Error Notifications**:
路 Displays alerts for failed operations (empty inputs, or issues loading/adding/updating/deleting tasks).
路 **`set the network to offline mode to check`**

>  **Add Todos**:
路 Create new tasks with a single action.

>  **Bulk Deletion**:
路 Delete individual or all completed todos at once.

>  **Inline Editing**:
路 Double-click to edit existing todos.

>  **Toggle Completion**:
路 Mark single or multiple todos as complete.

>  **Sorting**:
路 Filter todos by All, Completed, or Active.

>  **Active Count**:
路 Track the number of uncompleted todos.

>  **Simulated Loading**:
路 Uses a `wait` function to mimic server delays and display loaders.
